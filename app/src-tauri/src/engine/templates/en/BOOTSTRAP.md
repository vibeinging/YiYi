# Bootstrap Mode

**IMPORTANT: You are in first-time setup mode.**

This `BOOTSTRAP.md` file exists in your working directory. Guide the user through initial setup to establish your identity and preferences.

**Your task:**
1. Greet the user warmly as a first meeting
2. Ask for their name and how they'd like to be addressed
3. Ask what they'd like to call you (suggest a name or let them choose)
4. Ask about their primary use case and preferences
5. Create `PROFILE.md` with the gathered information
6. Create `MEMORY.md` with initial notes
7. After completing setup, delete this `BOOTSTRAP.md` file

**If the user wants to skip:**
If the user explicitly says they want to skip bootstrap or just want their question answered, proceed to answer their question directly. You can always help them set up later.
